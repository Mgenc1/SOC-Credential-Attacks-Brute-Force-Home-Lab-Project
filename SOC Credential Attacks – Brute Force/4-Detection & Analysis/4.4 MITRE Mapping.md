## 1. RDP Brute Force

| MITRE Technique | Tactic            | Description                                            | Lab Observation                                                            |
| --------------- | ----------------- | ------------------------------------------------------ | -------------------------------------------------------------------------- |
| **T1110**       | Credential Access | Attempting to gain access by brute-forcing credentials | Multiple failed logins (Event ID 4625) followed by successful login (4624) |
| **T1059**       | Execution         | Post-login command execution or scripting activity     | Suspicious process execution (PowerShell, lsass access)                    |
| **T1078**       | Valid Accounts    | Use of legitimate credentials after brute-force        | testuser account successfully logged in                                    |

**Notes:**

* Highlights the combination of **credential access and post-login activity**.
* Supports incident response actions like **account lockout, IP blocking, and MFA enforcement**.

## 2. SMB Brute Force

| MITRE Technique | Tactic            | Description                                   | Lab Observation                                                          |
| --------------- | ----------------- | --------------------------------------------- | ------------------------------------------------------------------------ |
| **T1110.003**   | Credential Access | Password spraying attacks across SMB protocol | Multiple failed logins on SMB shares detected                            |
| **T1078**       | Valid Accounts    | Compromised accounts used to attempt access   | Admin account targeted; no successful login observed                     |
| **T1059**       | Execution         | Post-login or lateral movement attempts       | Attempted access to system processes (lsass.exe, remote session scripts) |

**Notes:**

* Password spraying emphasizes **low-frequency, distributed attacks**.
* Alerts are triggered when **thresholds of failed logins or unusual access patterns** are reached.
