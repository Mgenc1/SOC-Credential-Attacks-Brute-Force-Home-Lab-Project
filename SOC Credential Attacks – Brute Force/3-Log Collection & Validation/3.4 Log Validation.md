##  1. Failed Logon Attempt (Event ID 4625)

```text
EventCode=4625
Account Name: testuser
Logon Type: 3
Source Network Address: 192.168.1.106
Failure Reason: Unknown user name or bad password
Status: 0xC000006D
Sub Status: 0xC000006A
````

###  Analysis

* Indicates **failed authentication attempt**
* Logon Type 3 → **Network logon (SMB / remote attack)**
* Source IP: **192.168.1.106 (attacker machine)**
* Repeated occurrences → **Brute force behavior**

```text
EventCode=4624

```

###  Analysis

* Confirms **attacker successfully guessed the password**

##  3. Important Note – Noise vs Real Signal

```text
EventCode=4624
Account Name: SYSTEM
Logon Type: 5
Process Name: services.exe
```

###  Explanation

* This is a **service logon (Logon Type 5)**
* Generated by Windows internally
* **NOT related to the attack**

SOC analysts must **filter these events out** to avoid false positives
