## 1️. Attack Environment

- **Attacker:** Kali Linux (192.168.1.106)  
- **Target:** Windows 11 host “umut” (192.168.1.107)  
- **Tools:** Nmap (port discovery), Hydra / Metasploit auxiliary module, PowerShell script (triggers Sysmon Event 1)  
- **Protocol:** RDP (TCP 3389)  

---

## 2️. Attack Steps

1. **Port Discovery**
   - Checked if RDP port (3389) is open on the target
   - Sysmon Event 3 logged network connection attempts

2. **Brute Force Attempts**
   The attacker attempted multiple username and password combinations against the RDP service.

**Attack Command Used:**


```bash
hydra -l testuser -P passwords.txt rdp://192.168.1.107
3. **Successful Login (Lab Scenario)**
   - Correct password found → Event 4624, Logon Type 3
   - Sysmon Event 10 logs post-login credential access attempt (`lsass.exe`)
```

### Command Explanation:

| Parameter             | Description                      |
| --------------------- | -------------------------------- |
| `hydra`               | Brute-force login cracking tool  |
| `-l testuser`         | Target username                  |
| `-P passwords.txt`    | Password wordlist used in attack |
| `rdp://192.168.1.107` | Target RDP service               |

## 4. Execution Narrative

> The attacker from IP 192.168.1.106 initiated a brute-force attack on the target host (192.168.1.107) via RDP. During the attempts, failed logons were recorded (Event 4625, Logon Type 3) for the user `testuser`. After a successful login (Event 4624), Sysmon Event 10 logged a credential access attempt targeting `lsass.exe`. All logs correlate and confirm the attack execution.
