### **1️. Incoming Alerts**

The SOC SIEM receives alerts based on:

- Multiple **failed login attempts** (Windows Security Event ID 4625)  
- **Unusual source IPs** attempting RDP connections  
- High frequency of logon attempts within a short timeframe

Example alert criteria:

| Alert Name                  | Trigger Condition                                      | Severity |
| ---------------------------- | ------------------------------------------------------ | -------- |
| RDP Brute Force Detected     | ≥5 failed login attempts from same IP within 2 minutes | Medium   |
| Suspicious RDP Source        | Login attempt from unknown or external IP             | High     |
| Post-Login Credential Access | Sysmon Event 10 on LSASS access after successful login | High     |

---

### **2️. Triage Steps**

1. **Alert Validation**  
   - Confirm the alert is **not a false positive** (e.g., legitimate admin logins).  
   - Check **source IP, username, and logon type** in the SIEM.

2. **Context Gathering**  
   - Identify targeted user accounts.  
   - Note event timestamps and frequency.  
   - Correlate **failed → successful login** events.

3. **Escalation Decision**  
   - If activity matches brute force indicators (multiple 4625 → 4624), escalate to **Tier 2/Deep Investigation**.  
   - If activity appears normal, **close the alert** with documentation.

---

### **3️. Triage Checklist**

| Step                         | Action                                                      | Notes                          |
| ---------------------------- | ----------------------------------------------------------- | ------------------------------ |
| Verify event type             | Check Event IDs 4625 / 4624                                 | Confirm failed vs successful  |
| Check Logon Type              | 3 = Network, 2 = Interactive                                 | RDP attacks are type 3         |
| Validate Source IP            | Compare against known corporate IPs                           | External IP → higher suspicion |
| Review Frequency              | Count attempts within time window                              | Threshold breach → escalate    |
| Document and Escalate         | Record findings in SIEM                                      | Attach logs and screenshots    |
