# BruteForce-Cases

This folder contains **two authentication attack scenarios** simulated in the SOC lab:

- **RDP Brute Force (Port 3389)**
- **SMB Brute Force (Port 445)**

Each attack scenario is documented in a separate file (`RDP Brute Force`, `SMB Brute Force`).  
This structure ensures that each attack can be analyzed independently, with its own logs, detection methods, and MITRE ATT&CK mappings.
