## 1. Identified Indicators of Compromise

Identified indicators were extracted from the collected Windows Security and Sysmon logs during the investigation. These artifacts can be used to detect similar malicious activities across the environment.

| IOC Type | Value | Description |
|--------|--------|-------------|
| Source IP | 192.168.1.106 | Attacker machine initiating authentication attempts |
| Destination IP | 192.168.1.107 | Target Windows host (`umut`) |
| Target Account | testuser | Account targeted during brute-force authentication |
| Logon Type | 3 (Network Logon) | Indicates remote network authentication attempt |
| Authentication Protocol | NTLM / NTLMv2 | Authentication mechanism used during login |
| Destination Port | 445 | SMB service targeted during the attack |
| Event ID | 4625 | Failed logon attempts detected |
| Event ID | 4624 | Successful authentication detected |
| Sysmon Event | 3 | Network connection telemetry confirming SMB communication |

### 2. Key Observations

- Multiple **Event ID 4625** logs indicate repeated failed authentication attempts.
- A **successful login (Event ID 4624)** from the same source IP suggests that the attacker eventually obtained valid credentials.
- **Sysmon Event 3** confirms network communication between the attacker host and the victim machine over **SMB (TCP 445)**.
- The attack originated from **192.168.1.106**, which should be investigated or blocked if malicious.

### 3. Recommended Detection Rules

Security teams can use the following indicators to detect similar attacks:

- Monitor repeated **4625 authentication failures** from the same source IP.
- Correlate **4625 followed by 4624** events within a short timeframe.
- Monitor suspicious **SMB authentication attempts (port 445)** from unusual hosts.
