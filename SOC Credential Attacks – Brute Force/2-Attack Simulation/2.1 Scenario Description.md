# Scenario Description

This lab simulates credential-based attacks observed in enterprise environments.

The scenarios focus on:

- RDP Brute Force
- SMB Brute Force

Detection is performed using both:

- **Windows Security Logs** (authentication events)
- **Sysmon Logs** (network and process activity)

---

## 1. Scenario – RDP Brute Force Attack

In this scenario, the attacker attempts to gain remote access via the **Remote Desktop Protocol (RDP)** service.

RDP (Port 3389) is commonly used for administration but is frequently targeted in brute force attacks.

During the attack:

- Multiple authentication attempts are sent to the RDP service
- Failed attempts generate **Event ID 4625**
- A successful attempt generates **Event ID 4624**
- **Sysmon Event ID 3** captures network connections to **port 3389**

This allows analysts to correlate authentication activity with actual RDP traffic.

---

## 2. Scenario – SMB Brute Force Attack

In this scenario, the attacker targets the **SMB service** on the victim machine.

SMB (Port 445) is widely used for file sharing and lateral movement.

During the attack:

- Repeated authentication attempts generate **Event ID 4625**
- A successful login generates **Event ID 4624**
- **Sysmon Event ID 3** records connections to **port 445**

This provides visibility into both authentication attempts and network-level behavior.

---

## 3. Expected Detection Signals

Across these scenarios, several indicators may appear:

### Authentication Indicators (Windows Logs)
- High volume of failed login attempts (4625)
- Successful login after failures (4624)

### Network Indicators (Sysmon)
- Connections to:
  - Port 3389 (RDP)
  - Port 445 (SMB)
- Repeated connections from a single source IP

### Correlation Indicators
- Same source IP across:
  - Failed logins
  - Successful login
  - Network connections

---

## 4. Key Detection Approach

Effective detection requires **log correlation** between:

- Windows Security Logs (authentication)
- Sysmon Logs (network + process activity)

This combination enables accurate identification of brute force attacks and reduces false positives.
